Welcome to Employee Directory

Employee directory is a sample app that demonstrates how to build modern web applications using different frameworks. This version is built with Backbone.js and Twitter Bootstrap.

Try It

Enter a few characters in the Search Box in the upper right corner of the screen, and select an employee. In the Employee view, you can navigate up and down the Org Chart by clicking either the Manager link, or one of the Direct Reports in the sidebar on the right of the screen.

Get the Code

The source code fot this application is maintained in this Github respository. The repository includes several adapters to use the application with different data persistance solutions: in-memory store, REST API, REST with JSONP, Node.js, MongoDB, etc.


Get More

The Empoyee Directory use case is simple enough to be easily understandable. It's also not entirely trivial and provides a good playground to test frameworks in a real-life context. Other versions of the application built with other frameworks ate available on the author's blog.
